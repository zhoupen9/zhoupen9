Libvirt Guest SSD

Table of Contents

Guest virtual machines may requires to know that provisioned disks are ssd or not, so that system runs trim command when discard unused disk spaces.

enable qemu tags

Disk types are not supported by libvirt yet, QEMU command line arguments required when using libvirt.

Add qemu:xxx tags mentioned in libvirt QEMU arguments.

get disk device id

for scsi disks, get disk device id line

  <address type="drive" controller="0" bus="0" target="0" unit="1"/>

the disk device id is scsi0-0-0-1.

Disk device can be sata or scsi.

add qemu override

add qemu command line arguments when provide disk 

  <qemu:commandline>
    <qemu:arg value='-set'/>
    <qemu:arg value='device.scsi0-0-0-1.rotation_rate=1'/>
  </qemu:commandline>

The qemu command line argument may not work due to 'no such device' error. When encountered this error, try override instead.

override qemu command line arguments 

  <qemu:override>
    <qemu:device alias="scsi0-0-0-1">
      <qemu:frontend>
        <qemu:property name="rotation_rate" type="unsigned" value="1"/>
      </qemu:frontend>
    </qemu:device>
  </qemu:override>
